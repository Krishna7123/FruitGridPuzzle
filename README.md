# Fruit Grid Puzzle

## Overview

Fruit Grid Puzzle is a 2D grid-based puzzle game developed in Unity. The player navigates through a grid, collects fruits to increase the score, and avoids bombs that reduce the score. The objective is to collect all fruits while maintaining a positive score.

---

## Features

- 2D Grid-Based Gameplay
- Random Fruit and Bomb Placement
- Player Movement using Arrow Keys
- Mobile Swipe Controls
- Score Tracking
- Move Counter
- Fruit Collection Counter
- Win, Lose, and Draw Conditions
- Restart Game
- Android APK Support
- User-Friendly UI

## Game Rules

- Each Fruit collected gives **+1 Score**.
- Each Bomb hit reduces **2 Score**.
- Collect all fruits to finish the game.
- If the final score is greater than 0, you Win.
- If the final score is 0, the game is a Draw.
- If the score becomes negative, the game ends with a Loss.

---

## How to Play

1. Start the game.
2. Move the player around the grid.
3. Collect all fruits.
4. Avoid bombs as much as possible.
5. Complete the level with a positive score to win.
